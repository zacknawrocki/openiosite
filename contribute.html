Contributors

We would love to see you contribute! If interested, please don't hesitate to email openiosnake@gmail.com. We will make sure to invite you to our Mattermost or Discord channels, direct you to helpful resources, and help you get involved in the Open IO community.

About Open IO

Open IO is a real-time multiplayer game engine, intended for the development of IO games using Node.js, Javascript, Socket.io, Express, and HTML Canvas. Currently, it consists of a working prototype, in which a snake-style IO game is implemented.
